Build a complete Enterprise Influencer Relations CRM & Workflow Management System.

Objective:
Create a centralized platform for managing clients, influencers, campaigns, approvals, tasks, reports, contracts, and performance tracking with full automation.

Technology:

* Frontend: Modern responsive web application
* Backend: Supabase
* Authentication: Supabase Auth
* Permissions: Row Level Security (RLS)
* Languages: Arabic (RTL) and English (LTR)
* Mobile responsive

========================================
USER ROLES & PERMISSIONS
========================

1. Admin

* Full system access
* Manage users
* Manage permissions
* View all reports
* Manage settings

2. Department Manager

* Approve campaigns
* Approve influencer lists
* View department KPIs
* Assign team leaders
* Approve budgets

3. Team Leader

* Manage team tasks
* Review influencer recommendations
* Follow campaign progress
* Submit approvals

4. Influencer Specialist

* Search influencers
* Create influencer recommendations
* Communicate with influencers
* Update campaign status

5. Account Manager

* Manage clients
* Create campaign requests
* Follow client approvals
* Generate reports

6. Finance User

* View budgets
* View campaign costs
* Generate invoices
* Manage payments

7. Client Portal User

* View shared campaigns only
* Review influencer recommendations
* Approve or reject influencers
* Add comments
